# ms-time

[![npm version](https://img.shields.io/npm/v/ms-time.svg)](https://www.npmjs.com/package/ms-time)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Tiny time duration parser and formatter (ms ↔ human string)

## Installation

```bash
npm install ms-time
```

## Functions

- `ms` — Convert human-readable string to milliseconds
- `formatMs` — Format milliseconds to human-readable string
